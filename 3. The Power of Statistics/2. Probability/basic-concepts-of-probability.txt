# Objective versus subjective probability
  + Probability use cases:
    - A company sill sell a certain amount of product in a 
      given time period
    - A financial investment will have a positive return
    - A political candidate will will an election
    - A medical test will be accurate
  + Types of Probability:
    - Objective 
      :- Based on statistics, experiments, and mathematical measerements.
      > Classical :- Based on formal reasoning about events with 
        equally likely outcomes.
        => CP = number of desired outcomes / total number of possible outcomes
      > Empirical :- Based on experimental or historical data
        => EP = number of times a specific event occurs / total number of events
    - Subjective 
      :- Based on personal feelings, experience, or judgement

# The basic rules of probability and events
  + Basic rules of probability
    - Complement rule
      . The event not occuring, in sense of opposite event will occur.
      P(A') = 1 - P(A)
    - Addition rule
      => Mutually exclusive events : two events are mutually exclusive if 
         they cannot occur at the same time.
         P(A or B) = P(A) + P(B)
    - Multiplication rule
      => Independent events : Two events are Independent if the occurrence of one event
         does not change the probability of the other event.
         P(A and B) = P(A) * P(B)
    . P(A) = probability of event A
    . P(B) = probability of event B
    . P(A') = probability of not event A

  + Types of events
