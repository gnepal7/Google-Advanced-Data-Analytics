+ Slice data 
 - df.iloc[]
   df.iloc[5:10, 2:] 	               → selects only rows 5 through 9, at columns 2+ 
   df.iloc[5:10] 		                 → selects only rows 5 through 9, all columns
   df.iloc[1, 2] 		                 → selects value at row 1, column 2
   df.iloc[[0, 2], [2, 4]] 	        → selects only rows 0 and 2, at columns 2 and 4
 - df.loc[]
   print(df)
   print()
   df.loc[:, ['color', 'class']]

# Histograms
  + Introduction to histograms
   - Histograms use bars to show data distribution, frequency, spread, center, and          possible outliers.
  + The importance of histograms
   - Histograms visualize data distribution, helping identify patterns, trends, and         outliers.
   - They support statistical analysis across finance, healthcare, engineering, and         social sciences.
  + How to interpret histograms
   1. Symmetric :- bell-shaped, showing data evenly distributed around the mean.
   2. Skewed :- A right-skewed histogram has a longer right tail and more left-side                  data points.
              A left-skewed distribution has a longer tail on the left side,                         indicating more data points on the right side.
   3.  Bimodal: has two distinct peaks, indicating that the data has two modes.
   4. Uniform: has a flat distribution, indicating that all data points are evenly                   distributed.
   - Use plt.hist() to create histogram
    plt.hist(df['seconds'], bins=range(40, 101, 5))
    plt.xticks(range(35, 101, 5))
    plt.yticks(range(0, 61, 10))
    plt.xlabel('seconds')
    plt.ylabel('count')
    plt.title('Old Faithful geyser - time between eruptions')
    plt.show();
