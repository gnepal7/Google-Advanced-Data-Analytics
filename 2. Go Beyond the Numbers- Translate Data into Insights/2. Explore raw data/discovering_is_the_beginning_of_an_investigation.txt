# Reference guide: Pandas methods for the discovery of a dataset
 - DataFrame.head() :- Displays the first n rows of the dataframe
 - DataFrame.info(X) :- Displays a summary of the dataframe
 - show_counts=True with other functions :- Returns the count of non-null Values
 - DataFrame.describe() :- Displays a descriptive statistics of entire dataset
 - df.shape :- Returns a tuple representing the dimensions of the dataframe
